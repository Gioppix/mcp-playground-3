# Epstein Email Explorer

An MCP server to search and explore 2322 Epstein emails.

**Try it:** `https://lively-poetry-gt8c1.mcp-use.run/mcp`

## [Read the full writeup](./REVIEW.md)

## Run locally

```bash
npm install
npm run dev
```

## Tools

| Tool                   | Description                            |
| ---------------------- | -------------------------------------- |
| `get-mentioned-people` | Get people mentioned in email bodies   |
| `get-participants`     | Get email senders and recipients       |
| `get-notable-figures`  | Get notable public figures mentioned   |
| `get-crime-types`      | Get all crime type categories          |
| `search-person`        | Search for a person across all indexes |
| `get-emails`           | Search emails by filters               |
| `show-emails` (**UI**) | Display emails in an email client UI   |
